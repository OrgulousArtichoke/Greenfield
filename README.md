# Greenfield

[![Stories in Ready](https://badge.waffle.io/OrgulousArtichoke/Greenfield.svg?label=ready&title=Ready)](http://waffle.io/OrgulousArtichoke/Greenfield)

- Node 0.12.5
- Websockets (latest)
- Phaser.io 2.0.5
- Socket.io 1.2.0
- Express 4.10.2
- JQuery 1.11.1
